# Estruturas Lineares

Repositório de estudos e exercícios sobre estruturas lineares e aplicativos web educativos.

## Conteúdo do repositório

- `Aula 01/`: exemplos em Python de vetores e esferas.
- `Aula 02/`: exercícios em Python sobre classes e estruturas de dados.
- `site/`: aplicação web de exemplo que calcula a média final de um aluno.
- `index.html`, `style.css`, `script.js`: versão principal da aplicação web na raiz do projeto.

## Tecnologias usadas

- Python
- HTML5
- CSS3
- JavaScript

## Descrição do projeto web

A pasta `site/` contém uma aplicação web simples para calcular a média entre duas notas parciais (M1 e M2) e indicar aprovação ou reprovação.

### Funcionalidades da aplicação

- Entrada de notas por meio de campos de formulário.
- Cálculo automático da média final.
- Exibição do resultado para o usuário.

## Como usar

1. Abra `index.html` ou `site/index.html` em um navegador.
2. Insira as notas de M1 e M2.
3. Clique em "Calcular Média".
4. Veja o resultado exibido na página.
