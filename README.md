# Boletim do Aluno

## Descrição

Este projeto é uma aplicação web simples e interativa que **calcula a média final de um aluno** baseado em duas notas parciais (M1 e M2).

## Funcionalidades

- **Entrada de Notas**: O usuário pode inserir a nota da M1 e M2 através de campos de input.
- **Cálculo de Média**: A aplicação calcula automaticamente a média final das duas notas.
- **Resultado**: Exibe ao usuário se foi aprovado (média ≥ 5) ou reprovado (média < 5) através de alertas visuais interativos.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização e layout
- **JavaScript**: Lógica de cálculo da média
- **SweetAlert2**: Biblioteca para alertas visuais personalizados

## Como Usar

1. Abra o arquivo `index.html` em um navegador web.
2. Digite a nota da M1 no primeiro campo.
3. Digite a nota da M2 no segundo campo.
4. Clique no botão "Calcular Média".
5. Receberá um alerta informando se foi aprovado ou reprovado.
